# Dash — Live Music Event Discovery

A futuristic dark-mode music event discovery web app built with React + Vite, deployed on Vercel.

## Live Demo
**[https://dash.vercel.app](https://dash.vercel.app)**

## Pages
- **Home** — Hero with animated soundwave, event preview, genre grid
- **Events** — Full list with live search and genre filters
- **Event Detail** — Ticket widget, attendance tracker, share options
- **Artists** — Artist profile cards
- **Venues** — Venue grid with capacity info

## Tech Stack
- React 18
- Vite 5
- Vercel (hosting + CI/CD)

## Local Development

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # production build
npm run preview   # preview production build
```

## Deploy
Vercel auto-deploys on every push to `main`.
